# Filled letters
> Allows you to create text with a fill image.

Quick creation and printing of text embedded in the image.

![](https://raw.githubusercontent.com/stasizon/filled-letters/master/screenshot.gif)

## Installation

```sh
npm i
npm start
```

## Development setup

```sh
npm i
npm run test
```
